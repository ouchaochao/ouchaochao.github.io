构建自己的Docker镜像

把一个Container commit成自己的

$ docker run -it centos
$ yum install -y vim
$ docker commit container_name cc/centos-vim

用dockerfile构建（推荐）

$ mkdir centos-vim 
$ cd centos-vim
$ vim Dockerfile

FROM centos
RUN yum install -y vim # 思考为什么image只读却可以写

$ docker build -t cc/centos-vim .
